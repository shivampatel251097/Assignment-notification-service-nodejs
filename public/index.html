This appliation is mainly backend application made with the NodeJs, Express, and some other services like Twilio, nodemailer, slack for send and receiving messages.

This page is documentation of how to use and navigate through this application.

Prerequisite for the same are: Postman, Slack, whatsapp, and phone number.

There are mainly 4 services are developed in the application: Mail, Text Message on Phone Number, Whats App messaging and message in the Slack channel

The services compact in the application are as follows:
  1. Send Mail to Subscribers Mail instantly.
  2. Schedule Mail to send to Subscribers Mail at scheduled time.
  3. Send Text message to Subscribers Phone Numbers instantly
  4. Schedule Text message to Subscribers Phone Numbers at scheduled time
  5. Send WhatsApp message to Subscribers WhatsApp Numbers instantly
  6. Schedule WhatsApp message to Subscribers WhatsApp Numbers at scheduled time
  7. Send Message on Slack channel instantly
  8. Send Message on Slack channel at scheduled time
Steps to use these services are documented in this file. You can download by clicking download button.
